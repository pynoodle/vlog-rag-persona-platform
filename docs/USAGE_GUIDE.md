# 🎭 PersonaBot 사용 가이드

## 🚀 GUI 실행 방법

### 현재 실행 중
Streamlit 서버가 **http://localhost:8502**에서 실행 중입니다!

### 브라우저 접속
1. 브라우저 열기
2. 주소창에 입력: `http://localhost:8502`
3. **Ctrl + Shift + R** (하드 새로고침)

---

## 📝 초기화 오류 해결

### 문제: 인코딩 오류
```
'charmap' codec can't encode character...
```

### 해결: ✅ 완료!
- `rag_manager.py`의 모든 print 문 수정
- 이모지 → 영문 표기로 변경
- safe_print() 함수 추가

### 다시 시도
1. 브라우저 **새로고침** (Ctrl + Shift + R)
2. 사이드바에서 **"🚀 시작하기"** 클릭
3. 약 20초 대기
4. ✅ 초기화 완료!

## 🎯 토론 시작 (4단계)

### 1️⃣ 주제 선택
4가지 토론 주제 중 하나 선택:
- 🌐 생태계 전쟁
- ✏️ S펜 제거
- 💰 가격 전략
- 🔮 폴더블 미래

### 2️⃣ 참가자 선택
**Galaxy (4명):**
- ☑️ 폴더블매력파 (564명)
- ☑️ 생태계딜레마 (37명, 좋아요 31)
- ☐ 폴더블비판자 (80명)
- ☐ 정기업그레이더 (58명)

**iPhone (3명):**
- ☑️ 가성비추구자 (8명, 좋아요 376!)
- ☐ Apple생태계충성 (79명)
- ☐ 디자인피로 (48명)

**직원 (3명):**
- ☑️ 마케터
- ☐ 개발자
- ☐ 디자이너

### 3️⃣ 설정
- 라운드 수: 1-3 선택
- 예상 메시지 수 표시
- 예상 시간 표시

### 4️⃣ 토론 시작
- **"🎬 토론 시작하기"** 버튼 클릭
- 실시간 진행 확인
- 결과 확인

## 🎨 UI 특징

### 아름다운 디자인
✨ **그라데이션 배경** (보라→분홍)

```.env
OPENAI_API_KEY=your-key-here
```
