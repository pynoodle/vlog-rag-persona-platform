## 🖥️ GUI 사용법

### 초기 화면
1. 사이드바에서 **"시스템 초기화"** 클릭
2. 20초 대기 (벡터 스토어 로딩)

### 토론 시작
1. **"토론"** 탭 선택
2. 토론 주제 선택 (4가지)
3. 참가자 체크박스 선택
4. **"토론 시작"** 버튼 클릭
5. 실시간 토론 진행 확인

### 페르소나 정보
- **"페르소나"** 탭: 7개 페르소나 상세 정보
- 규모, 좋아요, 대표 발언, 특징

### 결과 확인
- **"결과"** 탭: 토론 기록
- 메시지 내용, 통계, JSON 다운로드
