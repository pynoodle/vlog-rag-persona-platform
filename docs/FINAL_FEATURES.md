## ✨ 새로 추가된 기능

### 1️⃣ **전체 페르소나 지원** (7개) ✅

#### **고객 페르소나 (4개)**
```
✅ iPhone→Galaxy 전환자 (570명 데이터, 전환 강도 0.73)
✅ 갤럭시 충성 고객 (110명 데이터, 전환 강도 0.68)
✅ 기술 애호가 (분석형, 전환 강도 0.65)
✅ 가격 민감 고객 (가격 중시형, 전환 강도 0.55)
```

#### **직원 페르소나 (3개)**
```
✅ 마케터 (Samsung Mobile 시니어, 1,093명 분석)
✅ 개발자 (Android 앱 개발 리드, 기술 전문)
✅ 디자이너 (Product Design UX 리드, 사용자 중심)
```

---

### 2️⃣ **자유로운 토론 주제 입력** ✅

#### **3가지 입력 방식**

**방법 1: 추천 주제 선택**
```
📋 5개 추천 주제:
1. Galaxy Fold 7의 폴더블 혁신성이 충분한가?
2. 생태계 장벽을 극복할 수 있는 실질적 방안은?
3. 가격 프리미엄(100만원+)이 정당화될 수 있는가?
4. 30일 무료 체험 + 번들 할인 전략의 효과는?
5. iPhone 사용자가 Galaxy로 전환할 충분한 이유가 있는가?
```

**방법 2: 추천 주제 수정**
- 추천 주제를 선택한 후
- 텍스트 영역에서 자유롭게 수정

**방법 3: 완전히 새로운 주제 입력**
- "직접 입력" 선택
- 원하는 주제를 자유롭게 작성

---

### 3️⃣ **참가 에이전트 자유 선택** ✅

#### **고객 에이전트 선택**
```
□ iPhone→Galaxy 전환자
□ 갤럭시 충성 고객
□ 기술 애호가
□ 가격 민감 고객
```

#### **직원 에이전트 선택**
```
□ 마케터
□ 개발자
□ 디자이너
```

#### **자유로운 조합**
- 최소 1명 ~ 최대 7명
- 고객만 / 직원만 / 혼합
- 실시간 참가자 수 표시
- 예: "📊 선택된 참가자: **5명** (고객 3명 + 직원 2명)"

## 🚀 사용 시나리오

### **시나리오 1: 전체 팀 토론**
```
1. 모드: 🗣️ 자동 토론
2. 주제: "Galaxy Fold 7 출시 전략"
3. 참가자 선택:
   ✅ 모든 고객 페르소나 (4명)
   ✅ 모든 직원 페르소나 (3명)
4. 라운드: 3
5. 토론 시작!
→ 총 7명의 다양한 관점 확보
```

### **시나리오 2: 고객 중심 토론**
```
1. 모드: 🗣️ 자동 토론
2. 주제: "폴더블 폰 구매 시 최대 고민은?"
3. 참가자 선택:
   ✅ iPhone→Galaxy 전환자
   ✅ 갤럭시 충성 고객
   ✅ 가격 민감 고객
   ❌ 직원 페르소나 제외
4. 라운드: 2
5. 토론 시작!
→ 순수 고객 관점만 수집
```

### **시나리오 3: 전문가 분석**
```
1. 모드: 🗣️ 자동 토론
2. 주제: "Galaxy AI 기능 개발 우선순위"
3. 참가자 선택:
   ✅ 기술 애호가
   ✅ 개발자
   ✅ 디자이너
   ❌ 다른 페르소나 제외
4. 라운드: 5
5. 토론 시작!
→ 전문가 심층 분석
```

### **시나리오 4: 1:1 깊은 대화**
```
1. 모드: 💬 페르소나 채팅
2. 페르소나 선택: 디자이너
3. 질문: "폴더블 UI 디자인 시 가장 중요한 것은?"
→ 전문가와 심도 있는 대화
```

## 📊 페르소나 상세 정보

### **고객 페르소나**

#### **1. iPhone→Galaxy 전환자**
- **데이터:** 570명 전환 완료
- **전환 강도:** 0.73 (높음)
- **주요 관심사:**
  - ✅ 폴더블 혁신성
  - ✅ 화면 크기
  - ❌ 생태계 단절
  - ❌ UI 적응
- **대표 발언:** "폴더블 써보니까 진짜 신세계예요!"

#### **2. 갤럭시 충성 고객**
- **데이터:** 110명 폴더블 전문가
- **전환 강도:** 0.68 (높음)
- **주요 관심사:**
  - ✅ 폴더블 성숙도
  - ✅ 얇고 가벼움
  - ❌ S펜 제거
  - ❌ 가격 상승
- **대표 발언:** "Fold 3부터 써왔는데 7이 확실히 다릅니다"

#### **3. 기술 애호가**
- **데이터:** 분석형 사용자
- **전환 강도:** 0.65 (중상)
- **주요 관심사:**
  - ✅ 17 일반형 가성비
  - ✅ 정확한 정보
  - ❌ 스펙 차이 불명확
  - ❌ 가격 정당성
- **대표 발언:** "제가 계산해봤는데요, 17 일반이 압도적 가성비"

#### **4. 가격 민감 고객**
- **데이터:** 가격 중시형
- **전환 강도:** 0.55 (중)
- **주요 관심사:**
  - ✅ 할인 혜택
  - ✅ 가성비 모델
  - ❌ 높은 가격
  - ❌ 숨겨진 비용
- **대표 발언:** "50만원 차이면 다른데 쓰는게 낫죠"

### **직원 페르소나**

#### **1. 마케터**
- **역할:** Samsung Mobile 마케팅 팀 시니어
- **데이터:** 1,093명 전환 의도 분석
- **전문성:** 전략적, 데이터 중심
- **주요 관심사:**
  - ✅ 폴더블 차별화
  - ✅ 체험 마케팅
  - ❌ 생태계 장벽
  - ❌ 타겟 고객
- **대표 발언:** "전환 완료율 52.2%, 체험 마케팅이 핵심"

#### **2. 개발자**
- **역할:** Android 앱 개발 리드
- **데이터:** 기술 장벽, UI/UX 피드백
- **전문성:** 기술적, 구현 가능성
- **주요 관심사:**
  - ✅ Android 자유도
  - ✅ 개발 도구
  - ❌ 앱 호환성
  - ❌ 개발 비용
- **대표 발언:** "기술적으로 가능하지만, 개발 리소스가 관건"

#### **3. 디자이너**
- **역할:** Product Design 팀 UX 리드
- **데이터:** UI/UX 만족도, 디자인 선호도
- **전문성:** 사용자 중심, 경험 강조
- **주요 관심사:**
  - ✅ 폴더블 UI 혁신
  - ✅ 큰 화면 활용
  - ❌ 디자인 일관성
  - ❌ 학습 곡선
- **대표 발언:** "폴더블은 완전히 새로운 캔버스입니다"
